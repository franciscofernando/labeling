Enviroment(Entorno)

Lo principal en labeling es el atributo "env", este le da una identidad al elemento y lo transforma en un entorno.

Model(Modelo) & Bind(Lazo)

El modelo es espacio donde se guardan los atributos y/o metodos del entorno, se inicializa con el nombre del entorno y un objeto con los datos.

Dentro de un elemento(que contenga solo texto) se puede poner etiquetas bind(Lazos) de esta manera:
<p>{{nombreDelAtributo}}</p> o en un campo de texto se puede poner asi: <input type="text=" value="{{nombreDelAtributo}}" />

Controller(Controlador)

El controlador es una función que se ejecuta al existir el entorno, se inicializa con el nombre del entorno y una función.

Label(Etiqueta)

No solo el entorno puede ser identificado sino tambien los elementos con el atributo "label", esto sirve para usar el elemento de una manera mas facíl en el controlador.

Modificación de atributos

Si hay un lazo con un atributo, este cambia automaticamente si este cambia en el modelo.

Router(Enrutador)

El router es un objeto que vigila la url, y ejecuta diferentes eventos relacionados con esta.

Ejemplo:
router.export('/:variable', function(data){ alert('Ha cambiado la url y la variable es: '+data.variable); });

Change

El metodo principal de router, se usa para cambiar la url.

Ejemplo:
router.change('/admin'); //Usado con la explicación anterior devolvera un alert con: "Ha cambiado la url y la variable es: admin".

Router include

Es un comentario HTML con el formato: <!-- router NOMBRE URL:variable -->(:variable es opcional).

Ejemplo:
<!-- router usuario /user/:variable -->

Show

Este metodo imprime el html de la url de "Router include", como parametro opcional está el dar un objeto cuyos atributos se reemplazaran con las variables del include.

Ejemplo:
router.show('usuario', {variable: 1}) //incluirá la ruta HTTP /user/1

Hidden

Este metodo se utiliza para eliminar el contenido agregado por "router.show".

Ejemplo:
router.hidden('usuario'); //borrará todo lo agregado al ejecutar router.show('usuario')

Include

Se puede incluir el contenido de una url en el lugar donde esta el include, para esto hay que incrustar un comentario HTML, el include puede estar dentro de un entorno.

Ejemplo: <!-- include url.html -->
